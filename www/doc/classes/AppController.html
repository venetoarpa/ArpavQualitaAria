start

() Void

Defined in dev\controllers\appcontroller.js:27

Method to be run when the application first starts. It initializes our global (singleton-like) classes: - ModalHelper (used to display error and loading messages) - DataProvider and DataParser to retrieve data - The Router used to manage navigation between views - Settings and LocalStorage, to manage user settings - GeoHelper and MapHelper, adapters to the geolocation API and Leaflet API And loads the main view.
