geoLocalize

() Void

Defined in dev\prototypes\geohelper.js:69

Tries to geolocalize the user thanks to the HTML5 Geolocation API. If it succeeds, it updates class data (and whoever is listening to this class gets notified, as usual) and tries to find closest ozone and pm10, as well as updating the user marker position. Errors get logged to the console but nothing else happens, as geolocation is a best-effort function. The first call uses low precision methods, the second call tries to use high-precision methods if the devices has any.
