CentralinaAlertView Class

Extends Backbone.View
Defined in: dev\views\centralinaalertview.js:18
Module: controller

Renders buttons to toggle the alert system for the given station. Note that this does not inherit from the "magic" Marionette views, but from a bare backbone view, and has to provide a render method, and the latter has to be called explicitly.
